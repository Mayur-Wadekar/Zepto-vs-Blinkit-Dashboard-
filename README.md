Grocery Store War: Zepto vs Blinkit
Project Goal
This project contains an analysis of the competition between two major grocery delivery services: Zepto and Blinkit. The findings are presented in a Power BI report.

Project Files
grocery store war.pbix

This is the main Power BI report. It holds all the charts, analysis, and data model.

Zepto vs Blinkit For Power Bi.xlsx - Table2.csv

This file contains transactional data used in the report.

Zepto vs Blinkit For Power Bi.xlsx - Sheet1.csv

This file also contains transactional data used in the report.

Data Details
The data comes from the two CSV files. It includes details like:

Brand

Date

Product

Category

City

Price

Quantity

DeliveryTime(min)

PaymentMode

Rating

How to Get Started
What you need
You must have Power BI Desktop installed to open the report.

Steps to view the report
Download all project files to the same folder on your computer.

Open the file grocery store war.pbix using Power BI Desktop.

The report will automatically connect to the data in the CSV files.
